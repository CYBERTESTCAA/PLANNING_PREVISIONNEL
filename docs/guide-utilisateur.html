Se connecter a l'application

Authentification via votre compte Microsoft professionnel

Ouvrir le navigateur

Lancez Chrome ou Edge et allez a l'adresse https://192.168.13.51. Acceptez le certificat de securite si demande.

Fenetre de connexion Microsoft

Une fenetre popup Microsoft apparait automatiquement. Entrez votre adresse email professionnelle @caa... (la meme que pour Outlook et Teams).

Validation

Entrez votre mot de passe. Si l'authentification MFA est activee, validez sur votre telephone.

Acces a l'application

Vous etes connecte ! L'application charge automatiquement la page d'accueil (Planning). Votre nom s'affiche en haut a droite.

💡

Astuce : La session reste active plusieurs heures. Si vous fermez et rouvrez le navigateur, vous serez souvent reconnecte automatiquement sans re-saisir votre mot de passe.

⚠

Probleme de connexion ? Verifiez que vous etes bien connecte au reseau interne CAA (VPN si en teletravail). L'application n'est pas accessible depuis Internet.

Le Planning

Page principale — Grille d'affectation des equipes
/

C'est la page principale de l'application. Elle affiche une grille hebdomadaire avec les employes en lignes et les jours en colonnes. Chaque cellule montre les chantiers affectes et les absences.

La barre d'outils (en haut)

🏢

Selecteur de filiale

Choisissez votre filiale (ex: CAA Guadeloupe). Filtre tous les ateliers et employes.

🏭

Selecteur d'atelier

Choisissez un atelier dans la filiale. Ne montre que les equipes de cet atelier.

📅

Navigation temporelle

Fleches gauche/droite pour changer de semaine. Bouton "Aujourd'hui" pour revenir a la semaine en cours.

👁

Mode d'affichage

Basculer entre vue Semaine, Mois ou Annee. La vue annee donne une vision macro.

🔍

Palette de commandes

Raccourci Ctrl+K pour rechercher rapidement un employe, chantier ou action.

📈

Densite d'affichage

3 modes : Confort (grand), Compact, Ultra-compact. Adaptez selon votre ecran.

Affecter un employe a un chantier

Cliquer sur une cellule

Cliquez sur la case d'un employe pour un jour donne. La modale d'affectation s'ouvre.

Choisir le(s) chantier(s)

Selectionnez un ou plusieurs chantiers dans la liste. Chaque chantier a une couleur pour le differencier. Vous pouvez aussi choisir un OF specifique.

Choisir le creneau

Selectionnez Journee entiere, Matin (AM) ou Apres-midi (PM).

Valider

Cliquez sur "Affecter". La cellule se remplit avec le code couleur du chantier. L'affectation est sauvegardee immediatement.

Gerer les absences

Clic droit sur une cellule

Un menu contextuel apparait avec les options : ajouter une absence, supprimer une affectation, dupliquer...

Choisir le type d'absence

Types disponibles : Conge paye RTT Maladie Formation Autre

La cellule est marquee

Un badge colore indique l'absence. Impossible d'affecter un employe absent a un chantier ce jour-la.

Actions rapides (clic droit)

📅

Dupliquer sur demain

Copie l'affectation sur le jour suivant.

📄

Dupliquer sur la semaine

Copie l'affectation du lundi au vendredi.

🔄

Repeter chaque semaine

Repete l'affectation sur les 4 semaines suivantes.

🗑

Supprimer

Supprime l'affectation ou l'absence selectionnee.

Autres fonctionnalites du planning

Activez le toggle "Taux d'occupation" dans la barre d'outils. Un pourcentage et une barre de progression s'affichent pour chaque employe, montrant combien de jours il est affecte sur la periode visible.
Activez le "Mode validation" pour que chaque modification (ajout, suppression, duplication) genere une demande de modification au lieu de s'appliquer immediatement. Un responsable peut ensuite approuver ou rejeter chaque demande depuis le panneau dedie.
Boutons "Export Excel" et "Export PDF" dans la barre d'outils. Genere un fichier avec le planning de la semaine visible, incluant les employes, chantiers, absences et commentaires.
Bouton "Dupliquer semaine" : copie toutes les affectations de la semaine en cours vers la semaine suivante. Utile quand le planning est similaire d'une semaine a l'autre.
Boutons Annuler et Refaire dans la barre d'outils (ou Ctrl+Z / Ctrl+Y). Chaque action est enregistree et peut etre annulee. Historique limite aux 20 dernieres actions.
Activez les cases a cocher pour selectionner plusieurs employes a la fois. La barre d'actions groupees apparait : vous pouvez affecter un chantier ou une absence a tous les employes selectionnes en une seule operation.
Ouvrez le panneau "Activite" pour voir l'historique de vos actions : ajouts, suppressions, duplications. Utile pour retracer les modifications effectuees dans la session.
💡

Les jours feries sont automatiquement detectes et affiches en grise dans la grille. Ils ne comptent pas dans le taux d'occupation.

Vue Chantier

Voir le planning inverse : par chantier au lieu de par employe
/chantier

Cette page inverse la perspective : au lieu de voir "qui travaille quand", vous voyez "sur ce chantier, qui est affecte chaque jour".

Comment utiliser cette page

Selectionner la filiale et l'atelier

Comme pour le planning, choisissez d'abord votre filiale puis votre atelier.

Rechercher ou selectionner un chantier

Utilisez la barre de recherche pour filtrer par code ou libelle. Cliquez sur un chantier dans la liste de gauche.

Voir le planning du chantier

Le panneau de droite affiche une grille hebdomadaire : chaque ligne est un employe affecte, avec les jours en colonnes. Naviguez entre les semaines avec les fleches.

Informations affichees

👤

Employes affectes

Liste de tous les employes ayant au moins une affectation sur ce chantier.

📈

Statistiques

Nombre total de jours-homme, pourcentage de completion, dates de contrat.

🕐

Suivi du temps

Si la saisie des temps est activee : heures reelles vs heures prevues.

📌

Statut du chantier

A planifier En cours Bloque Termine

💡

Cas d'usage : Le chef d'atelier utilise cette vue pour verifier combien de personnes sont sur un chantier donne cette semaine et s'il manque des ressources.

Vue Gantt

Visualisation timeline de tous les chantiers sur plusieurs mois
/gantt

La vue Gantt affiche un diagramme de Gantt sur 6 mois. Chaque barre horizontale represente un chantier, avec sa duree (date de debut a date de fin de contrat) et le nombre de personnes affectees.

Comment lire le Gantt

📌

Barres horizontales

Chaque barre = un chantier. La longueur represente la duree prevue (date de contrat). La couleur correspond a la couleur du chantier.

👤

Nombre de personnes

Sur chaque barre, le nombre d'employes affectes est indique. Plus la barre est epaisse, plus il y a de monde.

📅

Ligne du jour

Une ligne verticale rouge indique la date du jour. Permet de voir en un coup d'oeil ou on en est.

🔍

Filtre par filiale/atelier

Memes selecteurs que le planning. Ne montre que les chantiers de l'atelier selectionne.

💡

Cas d'usage : Le responsable utilise cette vue pour avoir une vision globale de la charge sur les prochains mois et anticiper les periodes de sous/surcharge.

Calendrier Personnel

Vue calendrier individuelle d'un employe
/person-calendar

Cette page affiche le planning d'un seul employe sous forme de calendrier (vue mois, semaine, ou annee multi-mois). Chaque evenement montre le chantier affecte ou l'absence posee.

Comment utiliser

Selectionner filiale, atelier, equipe

Utilisez les selecteurs pour naviguer dans l'organisation. Cela filtre la liste des employes.

Choisir un employe

Dans le selecteur de personne, choisissez l'employe dont vous voulez voir le calendrier.

Consulter le calendrier

Le calendrier interactif affiche : les chantiers (colores selon le chantier) et les absences (en rouge/orange/bleu selon le type). Naviguez entre les mois avec les fleches.

Vues disponibles

📅

Vue Mois

Calendrier mensuel classique avec les evenements par jour.

🕐

Vue Semaine

Detail jour par jour avec les creneaux matin/apres-midi.

🗓

Vue Multi-mois

Vue annuelle sur 12 mois pour une vision globale de l'employe.

💡

Cas d'usage : Un chef d'atelier peut montrer a un employe son planning personnel prevu, avec ses conges et affectations, sur les prochains mois.

Plans / Bureau d'Etudes

Gestion complete des plans de fabrication par chantier
/plans

Cette page est le centre nevralgique du Bureau d'Etudes. Elle gere tous les plans de fabrication d'un chantier : creation, suivi de l'avancement, affectation aux dessinateurs, et gestion des articles/OF associes.

Selectionner un chantier

Menu deroulant en haut

Choisissez un chantier dans la liste. Vous pouvez taper le code ou le nom pour filtrer la recherche. Le menu est filtrable par la filiale selectionnee.

Les plans se chargent

Les plans du chantier s'affichent en tableau avec les statistiques globales (nombre de plans, % avancement).

Les 2 onglets

📄 Bureau d'Etudes

Tableau des plans avec toutes les colonnes editables :

  • Numero de plan (ex: 001, 002...)
  • Code HK (reference systeme)
  • Type de fabrication : Neuf, Reprise, Modif...
  • Etat d'avancement : A faire En cours A diffuser Valide A modifier
  • Etat usinage : Non commence, En cours, Termine
  • Dessinateur affecte au plan
  • Dates : echeance, diffusion, modification
  • Commentaires

📦 OF et Articles

Panneau lateral montrant les OF (Ordres de Fabrication) et leurs articles :

  • Liste des OF du chantier (depliables)
  • Articles dans chaque OF (code + designation + quantite)
  • Selection d'articles pour creer un plan
  • Association article ↔ plan existant
  • Badge indiquant si un article a deja un plan

Creer un plan

Methode 1 : Bouton "+ Nouveau plan"

Cliquez sur le bouton en haut. Remplissez le formulaire (numero, type, dessinateur, echeance). Le plan est cree vide et vous pouvez ensuite lui associer des articles.

Methode 2 : Depuis les articles

Dans le panneau OF, selectionnez un ou plusieurs articles avec les cases a cocher. Cliquez sur "Creer plan depuis la selection". Un plan est cree et les articles sont automatiquement rattaches.

Methode 3 : Creation groupee

Bouton "Creer depuis articles" pour creer plusieurs plans d'un coup a partir de la liste d'articles non encore planifies.

Modifier un plan

✏

Edition inline

Cliquez directement sur une cellule du tableau pour modifier la valeur. Sauvegarde automatique.

📄

Panneau d'edition

Cliquez sur le crayon d'un plan pour ouvrir un panneau lateral detaille avec tous les champs editables.

🗑

Supprimer

Bouton poubelle pour supprimer un plan. Confirmation demandee avant suppression.

👤

Gerer les dessinateurs

Bouton "Dessinateurs" pour ajouter, modifier ou desactiver des dessinateurs dans la liste.

Statistiques

En haut de la page, un resume en temps reel :

Total plans

Nombre de plans crees pour ce chantier

% Avancement

Repartition par etat (a faire, en cours, valide...)

Charge dessinateurs

Nombre de plans par dessinateur

⚠

Important : Les filtres "Recherche" et "Etat" en haut du tableau permettent de trouver rapidement un plan. Pensez a remettre les filtres a zero si vous ne trouvez pas un plan.

Questions Techniques

Suivi des questions et reponses par chantier (Kanban)
/questions

Cette page gere les questions techniques liees a un chantier. Les questions sont affichees en vue Kanban (3 colonnes) : Non traite, En cours, Termine.

Les 3 colonnes Kanban

🔴

Non traite

Questions en attente de reponse. A traiter en priorite.

🟠

En cours

Questions en cours de traitement. Une reponse partielle peut exister.

🟢

Termine

Questions resolues avec reponse complete.

Creer une question

Selectionner un chantier

Choisissez le chantier concerne dans le menu deroulant en haut de la page.

Cliquer sur "+ Nouvelle question"

Un formulaire apparait. Remplissez : la designation (titre court), le texte de la question detaillee.

Ajouter des pieces jointes (optionnel)

Vous pouvez attacher des photos ou documents (PDF, images). Cliquez sur le trombone ou glissez-deposez un fichier.

Valider

La question apparait dans la colonne "Non traite" en rouge.

Repondre et gerer

✏

Modifier la reponse

Cliquez sur une question pour l'ouvrir. Remplissez le champ "Reponse" et changez le statut.

✅

Changer le statut

Passez de "Non traite" a "En cours" puis "Termine" au fur et a mesure du traitement.

📎

Pieces jointes

Ajoutez ou consultez les pieces jointes : photos de chantier, plans PDF, etc. Apercu integre pour les images.

🗑

Supprimer

Bouton poubelle pour supprimer une question. Confirmation demandee.

💡

Astuce : Utilisez les questions pour formaliser les echanges techniques entre le Bureau d'Etudes et l'Atelier. Chaque question est tracee et consultable par tous.

Creer des OF

Regrouper les articles en Ordres de Fabrication avec des codes AA-ZZ
/creer-of

Cette page permet de regrouper les articles d'un chantier en Ordres de Fabrication en leur attribuant des codes de 2 lettres (AA, AB, AC... ZZ). Les articles avec le meme code sont automatiquement regroupes dans le meme OF.

Comment creer des OF

Selectionner un chantier

Choisissez le chantier dans le menu deroulant.

Voir la liste des articles

Un tableau affiche tous les articles du chantier avec : code article, designation, OF d'origine, quantite, et le code attribue.

Cliquer sur un article

Un panneau lateral s'ouvre avec une grille de codes (AA a ZZ). Recherchez ou cliquez sur le code voulu.

Repeter pour chaque article

Assignez un code a chaque article. Les articles avec le meme code seront dans le meme OF. Le numero d'OF se genere automatiquement.

Sauvegarder

Les codes sont sauvegardes automatiquement dans le navigateur (localStorage). Cliquez sur le bouton de sauvegarde pour enregistrer sur le serveur.

💡

Logique : Tous les articles ayant le code "AB" seront regroupes dans un meme OF. C'est une facon simple de creer des lots de fabrication par famille de pieces.

⚠

Brouillon local : Les codes sont d'abord sauvegardes localement dans votre navigateur. Pensez a sauvegarder sur le serveur avant de fermer la page si vous voulez que vos collegues voient vos modifications.

Synchronisation Fabric

Historique et declenchement manuel de la sync HK
/admin/sync
🔄

Lancer une sync manuelle

Bouton "Synchroniser maintenant" pour forcer l'import des donnees HK immediatement au lieu d'attendre le cron.

📋
